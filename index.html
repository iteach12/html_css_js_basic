CSS RESET

브라우저 기본 CSS스타일을 모두 초기화. 개발자의 의도를 정확히 구현하기 위함임. 마진, 패딩, 폰트크기, 굵기, 위치, 기본 색상 등 모든 것이 다 초기화됨. 코딩하기 전 세팅해 놓기~~

주석 스타일

  • 한 줄 주석 (ctrl+슬래시)
  • 여러 줄 주석 (블럭+ctrl+슬래시)
  • 주석은 항상 차곡차곡 쌓아둔다는 느낌으로

스타일 시트

종류 설명
브라우저 기본 스타일 웹 브라우저에 웹 문서를 표시할 때 브라우저에서 기본으로 사용하는 스타일. 이걸 안쓰려면, css reset 해주면 됨.
인라인 스타일 style 속성을 사용해 필요한 요소에 스타일을 직접 지정. html에서 바로 속성값으로 입력하는 방식. 급하거나, 반드시 이 방식이어야하는 경우가 아니라면 사용하지 말자...
내부 스타일 시트 문서 head에 넣어두고 이 문서에만 적용한다면 사용. 역시 필요한 경우에만 이렇게...
외부 스타일 시트 외부에 파일 생성하여 연결. css 파일끼리 import도 됨.

CSS 기본 선택자

종류 예시
전체 선택자 * { margin: 0;}
타입 선택자 p { font-style: italic;}
클래스 선택자 .bg { background-color:blue;}
id 선택자 #container { width:50px; }
그룹 선택자 h1, h2 { text-align:center; }
