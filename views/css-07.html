font-size

예전에는 절대 크기 단위인 px나 pt를 많이 사용했지만 모바일 기기까지 고려해야 하는 요즘에는 상대 크기 단위인 em이나 rem을 많이 사용함. 1em은 16px, 12pt와 동일.
종류 설명
em 부모 요소에서 지정한 글꼴의 대문자 M의 너비를 기준(1em)으로 한 후 비율값을 지정.
rem 문서 시작 부분(root)에서 지정한 크기를 기준으로 (1rem)한 후 비율값을 지정.
ex 해당 글꼴의 소문자 x의 높이를 기준(1ex)으로 한 후 비율값을 지정.
px 모니터 1픽셀을 기준으로 한 후 비율값을 지정.
pt 포인트라고 하며, 일반 문서에서 많이 사용.

텍스트 정렬 text-align

text-align 속성은 문단의 텍스트 정렬 방법을 지정.
종류 설명
start 현재 텍스트 줄의 시작 위치에 맞추어 문단을 정렬
end 현재 텍스트 줄의 끝 위치에 맞추어 문단을 정렬
left 왼쪽에 맞추어 문단을 정렬. 줄바꿈이 일어날정도로 텍스트가 많은 경우 오른쪽에 여백이 생길 수 있음. 왼쪽 기준으로 정렬시키기 때문임. 양쪽정렬 justify와의 차이점.
left, right 오른쪽에 맞추어 문단을 정렬
center 가운데 정렬
justify 양쪽에 맞추어 문단을 정렬. 왼쪽 정렬과 달리 줄바꿈이 일어날 때 오른쪽에 여백이 생기지 않게 꽉 채워서 텍스트가 표현됨
match-parent 부모 요소를 따라 문단을 정렬

none, underline, overline, line-through 옵션이 있음.

text-shadow

text-shadow : none | 가로거리 세로거리 번짐정도 색상
옵션 내용
가로거리 양숫값은 글자의 오른쪽, 음숫값은 글자의 왼쪽에 그림자 생성
세로거리 양숫값은 글자의 아래쪽, 음숫값은 글자의 위쪽에 그림자 생성
번짐정도 양숫값은 그림자가 모든 방향으로 크게 퍼짐. 음숫값은 그림자가 모든 방향으로 축소됨. 기본값은 0.
색상 그림자의 색상. 기본값은 글자색.
